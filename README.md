# TPS Preschool Management App

This repository contains the source code and documentation for the TPS Preschool Management App.

## Folder Structure

- `backend/`: Node.js API
- `mobile/`: Flutter application
- `docs/`: Project plans, database diagrams, and API documentation

## Branching Strategy

- **Main Branch**: `main` (Production-ready code only)
- **Development Branch**: `dev` (Primary development branch)
- **Feature Branches**: `feature/[feature-name]` (Branch off `dev`, merge back via Pull Request)

### Rules
- Never push directly to `main`.
- All work should be done in `dev` or feature branches.
- Merge to `main` only after thorough testing and approval.
